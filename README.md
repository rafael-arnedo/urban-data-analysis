
# Pipeline de Análisis Urbano – Movilidad vs Economía

## 📌 Problema
Analizar el impacto de la congestión vehicular en indicadores económicos en ciudades de LATAM.

## 🎯 Objetivo
Evaluar la relación entre movilidad urbana y variables económicas para generar insights que apoyen la toma de decisiones.

## ⚙️ Qué hice
- Construí un pipeline ETL en Python utilizando Pandas y NumPy
- Extraje y combiné datos de múltiples fuentes
- Limpié y transformé datasets para análisis
- Realicé análisis de correlación entre variables clave

## 📊 Resultados
- Identifiqué una correlación superior al **80%** entre población y PIB per cápita
- Detecté baja correlación (<30%) entre movilidad y PIB per cápita
- Generé insights orientados a decisiones en movilidad urbana

## 🛠️ Tecnologías utilizadas
Python | Pandas | NumPy | Jupyter Notebook | ETL | Análisis estadístico

## 📁 Estructura del proyecto
- `analysis.ipynb` → análisis completo
